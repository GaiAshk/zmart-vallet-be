# 💸 Smart Wallet Backend

A secure, lightweight backend for a Smart Wallet app built with **FastAPI**, **PostgreSQL**, and **JWT authentication**. Easily deployable with Docker and Kubernetes (Helm).

---

## 🚀 Features

- 🧾 Track expenses (title, amount, category, timestamp)
- 🔐 JWT-based user authentication
- 🐘 PostgreSQL integration
- 🐳 Docker + Docker Compose support
- ☸️ Kubernetes-ready with Helm charts
- 🧪 Testable with `pytest`

---

## 📦 Requirements

- Docker / Colima
- Python 3.13+
- PostgreSQL
- Make (optional)
